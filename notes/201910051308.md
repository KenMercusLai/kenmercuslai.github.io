# Git Tricks

1. Overwrite local files forcefully [^E010847D8D04]

```bash
git fetch --all
git reset --hard origin/master
git pull origin master
```

2. Migrate repository [^342A79E97D91]

```bash
git clone --mirror <repo>
cd <repo>
git remote set-url --push origin <new repo>
git push --mirror
```
