# HSRP study notes

Active, Standby routers:
* Active: the router forwards packets in a HSRP group 
* Standby: the router listens in a HSRP group

election rule:
priority (default is 100) > interface ip (bigger is better)

State machine:
* Initial
* Learn
* Listen

* Standby
* Active
